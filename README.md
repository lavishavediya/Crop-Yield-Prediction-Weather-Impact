# 🌾 Impact of Weather on Crop Yields: A Machine Learning Approach

## 🚀 Project Overview
This project is an end-to-end data analytics solution developed to predict agricultural productivity based on environmental factors. Using a dataset of 24,000+ records (1990-2013), I built a predictive system that achieves **97.76% accuracy**.

## 🛠️ The Data Pipeline
1. **Python:** Data cleaning, handling missing values, and merging datasets.
2. **SQL:** Structured storage and advanced querying for regional insights.
3. **Machine Learning:** Random Forest Regressor for high-precision yield forecasting.
4. **Power BI:** Interactive dashboards with "What-If" climate change simulations.

## 📂 Project Structure
- 📁 **01_Data_Cleaning:** Python scripts for preprocessing.
- 📁 **02_SQL_Queries:** Database schema and analytical queries.
- 📁 **03_ML_Model:** Model training, testing, and accuracy validation.
- 📁 **04_PowerBI_Dashboard:** Dashboard files and visual insights.

## 📊 Key Highlights
- **High Accuracy:** Model R² Score of **0.9776**.
- **Interactive Simulation:** A tool to predict yield changes by adjusting Temperature & Rainfall.
- **Global Scope:** Analysis covering 10 major crops across 100+ countries.
